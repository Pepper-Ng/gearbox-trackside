# gearbox-trackside

Spectator camera direction, automatic leaderboards, and printable per-driver telemetry reports for Gearbox Race Café's rFactor 2 simulator setup.

Current proof-of-concept notes live in `docs/core-poc.md`. The telemetry report PoC decision plan, including central-server versus rig-local collector testing, lives in `docs/telemetry-report-poc-plan.md`.
